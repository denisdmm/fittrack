# **App Name**: FitTrack Firestore

## Core Features:

- Autenticação do Usuário: Cadastro, login e gerenciamento de perfil de usuário seguros. Inclui a criação automática de um usuário administrador com login 'admin' e senha 'admin'.
- Controle de Grupo de Usuários: Gerenciamento de permissões e acesso baseado em grupos de usuários.
- Formulário de Criação/Edição de Usuários: Formulário para criar e editar usuários com campos: nome, sobrenome, email, login e senha (armazenada com bcrypt no Firestore).
- Biblioteca de Treinos: Navegue por uma coleção de rotinas de treino predefinidas com diferentes níveis de dificuldade e tipos de exercício.
- Criação de Treino Personalizado: Os usuários podem criar e salvar rotinas de treino personalizadas, adaptadas aos seus objetivos de condicionamento físico.
- Rastreamento de Treino: Registre os treinos concluídos, acompanhe o progresso e visualize as métricas de desempenho ao longo do tempo.
- Visualização de Progresso: Exiba o progresso do usuário usando tabelas e gráficos.
- Integração com Firestore: Armazene dados do usuário, rotinas de treino e progresso no Firestore.

## Style Guidelines:

- Cor primária: Azul celeste profundo (#00BFFF) para uma sensação vibrante e energética.
- Cor de fundo: Azul muito claro (#E0F8FF) para criar uma atmosfera limpa e calma.
- Cor de destaque: Ciano suave (#70dbdb), aproximadamente 30 graus 'à esquerda' do azul celeste profundo na roda de cores, fornecendo um elemento contrastante e complementar à interface do usuário.
- Fonte do corpo e título: 'PT Sans', sans-serif, para uma sensação moderna e amigável, adequada tanto para cabeçalhos quanto para o corpo do texto.
- Use ícones limpos e modernos para representar diferentes exercícios e categorias de treino.
- Um layout responsivo que se adapta a diferentes tamanhos de tela e dispositivos, proporcionando uma experiência de usuário consistente.
- Transições e animações sutis para aumentar o envolvimento do usuário e fornecer feedback durante as interações.
